createcv Design System

The complete visual language for the createcv AI resume builder — tokens, typography, and every interactive component with its states. Values mirror the live app (Tailwind config + index.css).

Logo & brand

Mark: a layered “file stack” (back sheets at low opacity) with a large AI sparkle. Wordmark: Futura, two-tone — create in ink, cv in primary blue.

Accent palette (resume)

User-selectable accent for the generated resume — 18 presets + a custom picker + an intensity control (40–100% opacity). Minimal & Classic templates ignore accent (kept black & white).

Derived tokens per accent: --accent (base), --accent-soft (10% tint), --accent-border (28%), --accent-strong (70%). Stored as concrete rgba() so they survive PDF export (html2canvas).

Accessibility

Baseline rules the UI must meet.

AreaRule
ContrastBody text ≥ 4.5:1. Ink on white = 13.6:1. Muted on white = 4.6:1. Primary on white = 4.6:1 (use for ≥14px/bold or large).
FocusEvery interactive element keeps a visible focus ring (3px rgba(37,99,235,.35)). Never remove outlines without a replacement.
TargetsTap targets ≥ 40×40px. Buttons/inputs are ≥ 40px tall.
MotionHonor prefers-reduced-motion; animations are decorative only.
ColorNever encode meaning by color alone — pair with a label/icon (ATS/PRO/FREE badges use text).
LabelsEvery input has a visible <label>; icon-only buttons use aria-label.
